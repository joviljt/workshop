What is HTML ?

HTML stands for Hyper Text Markup Language

HTML is the standard markup language for creating Web pages

HTML describes the structure of a Web page

HTML elements tell the browser how to display the content

HTML Code Example


<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<p>This is a paragraph.</p>

An HTML element is defined by a start tag, some content, and an end tag

What is CSS ?

CSS stands for Cascading Style Sheets

CSS describes how HTML elements are to be displayed on screen, paper, or in other media

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

CSS Code Example


body {
  background-color: lightblue;
}

h1{
  color: white;
  text-align: center;
}

p{
  font-family: verdana;
  font-size: 20px;
  color: white;
  text-align: center;
}

body/h1/p is a selector in CSS (it points to the HTML element you want to style:).

color is a property, and lightblue/white is the property value

text-align is a property, and center is the property value

What is Python ?

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

web development (server-side), software development, mathematics, system scripting, machine learning and etc....

Python was designed for readability, and has some similarities to the English language with influence from mathematics.

Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.

Python relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

Simple Data Analytics (15 min)


from datetime import datetime
import pandas as pd

# Sample data as a list of dictionaries
data = [
    {"Date": "2025-07-01", "Product": "Product A", "Revenue": 200},
    {"Date": "2025-07-01", "Product": "Product B", "Revenue": 150},
    {"Date": "2025-07-02", "Product": "Product A", "Revenue": 300},
    {"Date": "2025-07-02", "Product": "Product B", "Revenue": 250},
    {"Date": "2025-07-03", "Product": "Product A", "Revenue": 100},
    {"Date": "2025-07-03", "Product": "Product B", "Revenue": 400},
]

# Convert list to DataFrame
df = pd.DataFrame(data)
df['Date'] = pd.to_datetime(df['Date'])

# Show first few rows
print("📌 Preview of Data:")
print(df.head())

# Summary statistics
print("\n📊 Summary Statistics:")
print(df.describe())

# Total revenue overall
total_revenue = df['Revenue'].sum()
print(f"\n💰 Total Revenue: {total_revenue}")

# Total revenue by product
revenue_by_product = df.groupby('Product')['Revenue'].sum()
print("\n💼 Revenue by Product:")
print(revenue_by_product)

# Average revenue per day
average_daily_revenue = df.groupby('Date')['Revenue'].sum().mean()
print(f"\n📅 Average Daily Revenue: {average_daily_revenue:.2f}")

# Best-selling product
best_product = revenue_by_product.idxmax()
print(f"\n🏆 Best-Selling Product: {best_product}")

# Days with revenue > 500
high_revenue_days = df.groupby('Date')['Revenue'].sum()
print("\n🚀 High Revenue Days (>500):")
print(high_revenue_days[high_revenue_days > 500])

This Python code performs basic data analytics using a list of dictionaries to simulate sales data. It uses the Pandas library to convert the list into a DataFrame, converts date strings to datetime objects, and prints a preview of the dataset. It then calculates summary statistics such as total revenue, average daily revenue, and total revenue by product. The code also identifies the best-selling product and filters out the days where total revenue exceeds 500. This approach allows for quick insights from simple in-memory data without needing a CSV file or database.
